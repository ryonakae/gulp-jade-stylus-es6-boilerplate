# jade-stylus-es6-boilerplate
Boilerplate of jade, stylus and es6 by gulp.


## Require
* Node.js
* Gulp
* Bower


## Development Environment
### Task Runner
* Gulp

### HTML
* Jade

### CSS
* Stylus
  - kouto swiss

### JavaScript
* ES6
  - Browserify(Watchify)
  - Babelify
* Bower
  - jQuery
  - Velocity.js

### Local server
* BrowserSync


## Commands
### Download

* From master: [Download](https://github.com/ryonakae/jade-stylus-es6-boilerplate/archive/master.zip)
* From release: [Releases](https://github.com/ryonakae/jade-stylus-es6-boilerplate/releases)

### Install npm packages

    $ npm i
    
### Install bower

    $ bower i

### Development build

    $ gulp
    
### Production build

    $ gulp --env production
